RadioManager Library Documentation

Documentation for the RadioManager library and its dependencies

RadioManager Class

CONSTRUCTOR RadioManager(uint8_t ce_pin, uint8_t csn_pin, const char* radio_id)

Creates a new RadioManager instance.

Parameters:

ce_pin - CE pin for the nRF24L01 module*required
csn_pin - CSN pin for the nRF24L01 module*required
radio_id - Unique identifier for this radio (will be trimmed to 4 characters)*required

Public Functions

Operations

PUBLIC bool begin()

Initializes the radio module and configures initial parameters.

Returns:

true if initialization was successful, false otherwise
PUBLIC void loop()

Main function to be called frequently in the program's main loop. Manages the different states of the RadioManager.

States:

IDLE           - Ready to send/receive messages
TRANSMITTING   - Currently sending a message
RECEIVING      - Currently receiving a message
PAIRING_LISTEN - Listening for pairing requests
PAIRING_TRANSMIT - Transmitting pairing information
PUBLIC State getCurrentState()

Gets the current state of the RadioManager.

Returns:

The current state of the RadioManager
PUBLIC bool isBusy()

Checks if the RadioManager is busy (pairing, transmitting or receiving).

Returns:

true if the RadioManager is busy, false otherwise

true if the RadioManager is available, false otherwise
